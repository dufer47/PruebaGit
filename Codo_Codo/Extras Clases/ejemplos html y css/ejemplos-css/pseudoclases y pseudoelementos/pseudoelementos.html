Pseudoelementos

Un pseudoelemento en CSS es un término utilizado para describir una parte específica de un elemento HTML que puede ser seleccionada y estilizada independientemente del contenido real del elemento. Los pseudoelementos se crean utilizando dos puntos dobles (::) seguidos de un nombre específico, como ::before o ::after. Los pseudoelementos permiten agregar contenido o estilos a partes de un elemento HTML sin tener que modificar el marcado HTML real.

::first-letter se utiliza para darle estilo a la primera letra de un texto.

::first-line se utiliza para darle estilo a la primera línea de un párrafo.

La pseudoclase :first-line en CSS se utiliza para seleccionar y aplicar estilos específicos a la primera línea de texto dentro de un elemento.

Esta pseudoclase permite estilizar la primera línea de un bloque de texto de manera independiente, lo que puede ser útil para crear efectos tipográficos o de diseño específicos para esa parte del contenido.

::selection agrega estilos a una parte del documento que ha sido resaltada por el usuario.
