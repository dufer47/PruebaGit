Selectores avanzados

Un selector avanzado en CSS se refiere a una técnica que permite seleccionar elementos específicos en una página web utilizando una combinación de selectores más compleja que simplemente seleccionar elementos por su etiqueta, clase o identificador. Los selectores avanzados son parte fundamental de la potencia y flexibilidad de CSS para aplicar estilos y dar formato a páginas web.

Selector agrupado

Utiliza como caracter de separación la , (coma) y permite aplicar estilos a varios selectores al mismo tiempo. Por ejemplo, puedes aplicar el mismo estilo a múltiples clases: .clase1, .clase2, .clase3 { estilo: valor; }.

Selector descendiente

Utiliza como caracter de separación el espacio y permite seleccionar un elemento que es descendiente de otro elemento. Por ejemplo, puedes seleccionar todos los párrafos < p > dentro de un div con una clase específica: div.mi-clase p.

Párrafo con clase elemento-descendiente, que coloca el texto en cursiva. El selector descendiente le aplica el resto de los estilos, a aquellos elementos que tengan este selector y estén dentro de un < div >.

Div con clase elemento-descendiente, que coloca el texto en cursiva. El selector descendiente le aplica el resto de los estilos, a aquellos elementos que tengan este selector y estén dentro de un < div >.

Encabezado de nivel 3 con clase elemento-descendiente, que coloca el texto en cursiva. El selector descendiente le aplica el resto de los estilos, a aquellos elementos que tengan este selector y estén dentro de un < div >.

Párrafo sin la clase elemento-descendiente, a pesar de estar dentro del < div > no se le aplican los estilos del selector descendiente.

Selector de hijos directos

Utiliza como caracter de separación el > (mayor) y permite seleccionar elementos que son hijos directos de otro elemento, descartando nietos y sucesivos. Por ejemplo, puedes seleccionar todos los elementos li que son hijos directos de una lista desordenada < ul >: ul > li.

Selector de hermano adyacente

Utiliza como caracter de separación el + (signo más) y permite seleccionar un elemento que es adyacente y tiene el mismo elemento principal (el mismo "padre") que otro elemento específico. En otras palabras, selecciona el elemento inmediatamente después del elemento de referencia en la estructura HTML si ambos comparten el mismo elemento padre.

Selector general de hermanos

Utiliza como caracter de separación la ~ (virgulilla o tilde de la ñ) y permite seleccionar todos los elementos que son hermanos del mismo nivel y que siguen al elemento de referencia en la estructura HTML, siempre que compartan el mismo elemento padre.
