Transiciones

Una transición en CSS es una técnica que permite crear efectos suaves y animaciones entre dos estados diferentes de un elemento HTML. Estas transiciones son útiles para mejorar la experiencia del usuario al proporcionar una transición gradual y visualmente atractiva entre los estilos de un elemento antes y después de que ocurra un cambio, como un evento de hover (pasar el cursor sobre un elemento) o un cambio de estado. Las transiciones se aplican a las propiedades CSS de un elemento, como el color, el tamaño, la posición o cualquier propiedad que pueda cambiar con el tiempo. Cuando se activa un evento, como el paso del cursor sobre un elemento, la propiedad cambia de un valor a otro de manera suave y gradual en lugar de cambiar instantáneamente.
