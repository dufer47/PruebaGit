Pseudoclases

Una pseudoclase en CSS es una palabra clave que se usa para seleccionar un estado o comportamiento específico de un elemento HTML. Las pseudoclases permiten aplicar estilos a elementos en función de su interacción con el usuario o su posición en la estructura del documento, sin necesidad de agregar clases o atributos adicionales en el marcado HTML. Las pseudoclases se escriben con dos puntos (:) seguidos de la palabra clave que describe el estado o comportamiento que se desea seleccionar.

Pseudoclases para hipervínculos:

Las pseudoclases para hipervínculos en CSS se utilizan para aplicar estilos a enlaces o anclas < a > en función de su estado o comportamiento específico. Estas pseudoclases son especialmente útiles para mejorar la experiencia del usuario al resaltar enlaces cuando el usuario interactúa con ellos.

  • :link se refiere a un enlace que todavía no ha sido visitado.
  • :hover se refiere a un elemento sobre el que se coloca el puntero del mouse.
  • :visited se refiere a un enlace que ya ha sido visitado.
  • :active se refiere a cualquier elemento que ha sido activado por el usuario.
